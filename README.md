# Student FAQ API using WSO2 API Manager

## Description
This project demonstrates how to use WSO2 API Manager in a real-world application.

- Backend: PHP (WAMP Server)
- Data Source: JSON file
- Frontend: HTML, Bootstrap, JavaScript
- API Management: WSO2 API Manager
- Security: OAuth2 (Bearer Token)

## How it Works
1. PHP file serves FAQ data from JSON
2. WSO2 API Manager exposes it as a REST API
3. Frontend fetches data using secured API with access token

## Features
- Dynamic FAQ loading
- Search functionality
- API security using OAuth2
- Clean UI using Bootstrap

## Note
This project runs on localhost. Screenshots are provided to demonstrate functionality.
API secured using OAuth2 Bearer Token via WSO2 API Manager
